# Rationale

What does it means liquid models?
In 2026 raise of open weitghts models and cheap Chinese inference provider are distrupting the lock in OpenAI and Anthropic tried to perform in the last period.

Company can run open weights model on premise if they want more privacy.

For these reasons, it is increasing becoming important to be able to switch between models.

Also, reviewing the work of a model with a completly different one proves to be effective (i.e. Review Anthorpic's  Fable with GPT-Sol).

## Code+Markdown is the memory

To be able to switch between models, the code and the markdown files need to be the only source-of-truth.

This project aims to select at least one Software Development Lifecycle (SDLC) library already in place.

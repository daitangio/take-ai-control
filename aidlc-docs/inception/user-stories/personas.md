# User Personas — nello

## Persona 1: Board Owner (Alice)
- **Role**: Creates and owns boards
- **Goals**: Organize work visually, invite teammates, track progress
- **Behaviors**: Creates boards, adds lists/cards, assigns cards to others, monitors activity

## Persona 2: Collaborator (Bob)
- **Role**: Invited to a board by the owner
- **Goals**: See assigned cards, update card status, move cards across lists
- **Behaviors**: Logs in via magic link, views shared board, drags cards, edits card details

## Persona 3: New User (Carol)
- **Role**: First-time user with a valid email domain
- **Goals**: Gain access quickly without password friction
- **Behaviors**: Enters email, receives magic link, clicks link, lands on empty board list
